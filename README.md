# 🎤 藝人新聞搜尋系統

這是一個使用 **Python Flask** 打造的新聞搜尋網站，  
可即時查詢特定藝人於多家新聞媒體上的最新報導。

---

## 🚀 功能特色
- 🔍 支援藝人關鍵字搜尋（共 35 位藝人）
- 📰 搜尋來源包含：
  聯合報、中國時報、鏡新聞、自由時報、壹蘋新聞網、ETtoday、三立新聞網、Yahoo新聞、TVBS新聞網、CTWANT、LIFE生活網、Nownews、車勢新聞、噓星聞、其他新聞
- 📅 支援時間範圍選擇：
  - 當天
  - 當週（週一～今日）
  - 當月（1 號～今日）
- ✅ 可選擇特定新聞媒體或全選
- 💾 即時顯示搜尋結果（標題、來源、連結）

---

## 🛠 技術架構
- 前端：HTML + CSS + JavaScript
- 後端：Python Flask
- 爬蟲：requests + BeautifulSoup
- 資料格式：JSON
